Protein Analysis Tool

The program calculates:

  • Molecular weight
  • Amino acid composition
  • pI
  • Extinction co-efficient
  • Protein concentration from inputed A280

You can also enter a DNA sequence for translation and analysis!

Welcome to the Protein Analysis Tool web app for analysing protein sequences. Simply select sequence type, enter your sequence and click "Submit".



Use single letter code for proteins



You can successively enter multiple OD values if required.
